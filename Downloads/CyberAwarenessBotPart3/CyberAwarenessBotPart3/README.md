# 🛡️ CyberAwarenessBot – Part 3

A WPF-based Cybersecurity Awareness Chatbot built in C# with Visual Studio 2022.

## Features

### 1. 💬 Chat with NLP Simulation
- Type naturally about cybersecurity topics
- Understands different phrasings (e.g. "2FA", "two-factor", "multi-factor")
- Fallback responses for unrecognised input

### 2. 📋 Task Assistant
- Add cybersecurity reminder tasks with title, description, and reminder date
- View all tasks in a list
- Mark tasks as complete or delete them

### 3. 🎮 Cybersecurity Quiz
- 12 questions covering phishing, passwords, 2FA, malware, and social engineering
- Multiple-choice and true/false formats
- Immediate feedback with explanations
- Final score with motivational message

### 4. 📜 Activity Log
- Records every action taken by the bot with timestamps
- Viewable and clearable from the Activity Log tab

## Technologies Used
- C# / WPF (Windows Presentation Foundation)
- .NET 8.0
- XAML
- ObservableCollection for live data binding

## How to Run
1. Clone the repo
2. Open `CyberAwarenessBotPart3.sln` in Visual Studio 2022
3. Press `Ctrl + Shift + B` to build
4. Press `F5` to run
